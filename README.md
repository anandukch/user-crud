# User Crud api

## Base url
### `http://localhost:3000` 
## Base route
### `/api/user` 
## Sub routes
### `/create`
### `/get/:userid`
### `/update/:userid`
### `/delete/:userid`


## Note
### Add a `.env `file to the root of your project with the following content:
  `
  MONGO_URL=<mongo url>
  `
